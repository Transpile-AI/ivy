The `ivy_tests` folder contains test-related files for the Ivy library, an open-source machine learning computations library.

Files in the `ivy_tests` directory:
------------------------------------
1. `ivy_tests/__init__.py`:
   - This is an empty Python file that signifies the folder is a Python package.

2. `ivy_tests/conftest.py`:
   - This file contains configuration settings and fixtures that are shared across multiple test files within the Ivy testing framework.

3. `ivy_tests/test_docstrings.py`:
   - This Python file contains tests specifically focused on checking the validity and correctness of docstrings used in the Ivy library. These tests ensure that the docstrings provide accurate and helpful information about Ivy's functionalities.
